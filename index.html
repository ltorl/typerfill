Install it

  1. Click Download for Chrome above to get the zip.
  2. Unzip it — you'll get a typerfill folder.
  3. Open chrome://extensions in Chrome.
  4. Turn on Developer mode (top right).
  5. Click Load unpacked and select the unzipped typerfill folder.
  6. That's it — start typing in any text field on the web.
Typerfill is free and open source.
